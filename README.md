## Deskripsi

Aplikasi Pencatat Keuangan adalah aplikasi desktop berbasis GUI yang dikembangkan menggunakan Python dan PySide6 untuk membantu pengguna dalam mencatat serta mengelola pemasukan dan pengeluaran secara sederhana. Aplikasi ini memungkinkan pengguna untuk menyimpan data transaksi ke dalam database SQLite, menampilkannya dalam bentuk tabel, serta menghitung total saldo secara otomatis sehingga memudahkan pemantauan kondisi keuangan.

## Fitur Utama

* Menambahkan data transaksi 
* Mengedit data transaksi
* Menghapus data transaksi
* Menampilkan data dalam tabel
* Perhitungan total saldo otomatis
* Validasi input jumlah hanya angka
* Pemilihan tanggal menggunakan 
* Kategori menggunakan dropdown 
* Tampilan tabel dengan warna berbeda untuk pemasukan & pengeluaran


## Cara Menjalankan Aplikasi

 Python sudah terinstall di perangkat  (minimal versi 3.8), kemudian install library yang dibutuhkan dengan menjalankan perintah pip install PySide6. Setelah itu, jalankan aplikasi dengan perintah python main.py melalui terminal atau command prompt, dan aplikasi akan terbuka serta siap digunakan.

## Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan Python 3 sebagai bahasa pemrograman utama, PySide6 sebagai framework untuk membuat antarmuka GUI, SQLite sebagai database untuk menyimpan data transaksi, serta QSS (Qt Style Sheets) untuk mengatur tampilan antarmuka agar lebih menarik.
